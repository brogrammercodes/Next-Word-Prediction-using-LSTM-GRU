# 🧠 Next Word Prediction using LSTM & GRU

## 📌 Overview
This project implements deep learning models using LSTM (Long Short-Term Memory) and GRU (Gated Recurrent Unit) architectures to predict the next word in a sequence.

The system learns contextual patterns in text data and generates accurate next-word predictions, making it useful for applications like autocomplete, chatbots, and text generation.

---

## 🎯 Problem Statement
Traditional models fail to capture long-term dependencies in sequential text data.

This project solves that by:
- Using LSTM and GRU to capture context and sequence dependencies  
- Learning patterns from large text corpora  
- Generating meaningful next-word predictions  

---

## 🚀 Features
- Text preprocessing and tokenization
